# Telstra Messaging App

This is a web-based messaging application that allows users to send and receive messages using the Telstra API. It is built with Next.js, TypeScript, and Tailwind CSS.

## Project Structure

- `app/`: Contains the application pages and routing.
- `components/`: Contains the UI components.
- `hooks/`: Contains custom React hooks.
- `lib/`: Contains the core application logic and services.
- `public/`: Contains static assets.
- `styles/`: Contains global styles.
