This directory contains the core application logic and services for the Telstra Messaging App. It is responsible for handling business logic, data access, and communication with external services.

- `auth.ts`: Handles user authentication and authorization.
- `contact-service.ts`: Manages user contacts.
- `database.ts`: Contains database connection and query logic.
- `inbox-service.ts`: Manages user message inboxes.
- `media-service.ts`: Handles media uploads and downloads.
- `messaging-service.ts`: Core service for sending and receiving messages.
- `reports-service.ts`: Generates reports and analytics.
- `rules-service.ts`: Manages user-defined messaging rules.
- `telstra-api.ts`: Interacts with the Telstra API for messaging services.
- `template-service.ts`: Manages message templates.
- `types.ts`: Defines custom TypeScript types and interfaces used throughout the application.
- `utils.ts`: Contains utility functions used across the application.
- `websocket-server.ts`: Manages the WebSocket server for real-time communication.
