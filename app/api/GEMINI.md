This directory contains the API routes for the application, built with Next.js API Routes. These routes handle requests for authentication, messaging, and other backend services.

- `account/`: API routes for managing user accounts.
- `admin/`: API routes for admin-specific functionality.
- `auth/`: API routes for user authentication.
- `messaging/`: API routes for sending and receiving messages.
- `socket/`: API route for initializing the WebSocket server.
